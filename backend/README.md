<h1>DFD - Backend</h1>

<h2>Tech Stack</h2>
<ul>
    <li>Node 18</li>
    <li>Express</li>
    <li>PosgresSql 15.6</li>
    <li>docker</li>
</ul>

<h2>How to start local server</h2>
 - RUN `yarn install`
 - RUN `docker-compose up -d`
 - RUN `yarn start`

NOTE: database will be set up on `server start` and will run in docker container. For `.env` config
